# Ailurus Web

Ailurus 创作者订阅平台 — UI 原型（Sui Overflow 2026）

## 启动

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 功能（UI 原型）

- 路由：Home / Explore / Create / Profile / Creator / Wallet / Notifications / Settings
- 弹窗：Google 登录、USDC 充值、订阅、上传、帖子详情、成为创作者
- Toast：Sonner 操作反馈
- 响应式：移动端底部导航 + 桌面端侧边栏

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS 4
- React Router 7
- Sonner (toast)
- Lucide React (icons)

## 说明

当前为纯前端 UI 原型，链上功能尚未接入。项目说明见根目录 [PROJECT.md](../PROJECT.md)。
